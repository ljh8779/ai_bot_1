# 비전문가용: 우리 사내 RAG 봇, 그림으로 이해하기

## 1. 30초 요약
- 이 프로그램은 **회사 문서를 읽어두고**, 질문이 들어오면 **관련 문서 조각을 찾아서** 답변해주는 AI입니다.
- 사용자 부서/권한에 맞는 문서만 보여주도록 설계되어 있습니다.
- 답변 시 출처 조각도 함께 제공합니다.

## 2. 문서를 넣을 때(수집)
1. 텍스트나 파일(PDF/PPTX 등) 업로드
2. 문서를 청크로 분할
3. 각 청크를 임베딩(숫자 벡터)으로 변환
4. DB에 저장
5. 중복 문서는 content hash로 방지

## 3. 질문할 때(답변)
1. 사용자 질문 입력
2. 유사한 문서 청크 검색
3. 사용자 부서/역할 ACL 필터 적용
4. AI가 근거 청크 기반으로 답변 생성
5. 답변 + 출처 반환

## 4. 안전성
- 문서별 `allowed_departments`, `allowed_roles` 메타데이터 적용
- 권한 없는 문서는 검색 결과에서 자동 제외
- 접근 가능한 근거가 없으면 안전한 안내 응답 반환

## 5. 화면에서 하는 일
1. 문서 등록
2. 질문 테스트
3. 근거 확인

## 6. 용어 설명
- RAG: 문서를 먼저 찾고 그 문서를 근거로 답하는 방식
- 임베딩: 텍스트를 숫자 벡터로 변환
- 청크: 긴 문서를 작은 단위로 나눈 조각
- 출처 조각: 답변에 실제 사용된 문장 일부
